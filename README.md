Urban Threads boutique
Urban-threads-boutique Website
 Project Overview Urban Threads Boutique is a multi-page website created as part of my web development coursework at SEBANCOLLEGE.
The project showcases a fictional fashion brand specializing in clothing design, consultation, and seasonal collections.

Features

Home Page – Attractive landing page with brand introduction and service highlights.
About Page – Background and mission of Urban Threads Boutique.
Services Page– Detailed list of offerings (custom designs, consultations, collections, alterations).
Contact Page – Contact details including email, phone, and address.
Enquiry Page – Simple HTML form for customer enquiries.
Site Map Page – Overview of the website structure with links to all pages.
Setup Instructions

Save all HTML files (index.html, about.html, services.html, contact.html, enquiry.html, sitemap.html) in the same folder.

Place the logo image (logo.png) in the same folder or inside an images folder.

Open index.html in any web browser to view the site.

Use the navigation links to explore the different pages.

Usage

Visitors can browse the Home page to see what the boutique offers.
Navigate to the About page to learn more about the brand.
Explore the Services page for detailed offerings.
Use the Contact page for direct communication details.
Submit questions or requests through the Enquiry page.
View the Site Map page for a clear overview of the site’s structure.
